Configuraciones iniciales

 Top  Previous  Next

Antes de comenzar a trabajar con la aplicación es necesario realizar algunas configuraciones.

 

1.Primero se debe conectar el sistema móvil con la web, para que se sincronicen y se puedan descargar los datos necesarios al teléfono.

 


Primera configuración

 

Luego de conectar el teléfono a la red Wifi, introduzca en el campo de texto la dirección del sitio web. Por ejemplo: http://www.horusmb.cu:8000 o http://192.168.63.1:8000/HorusMB, y presione Continuar.

 


Introducir url del sistema web

 

 

2. Después de conectarnos al sistema web, es necesario por seguridad introducir credenciales, estas son las mismas que se utilizan para acceder la web. Luego presiona Continuar.

 


Introducir credenciales

 

3. Presiona en el botón Sincronizar para descargar los datos.

 


Descargar datos

 

 

Luego que se carguen los datos. Presione Continuar.

 


Información

 

4. Por último seleccione el área de su preferencia y presione Continuar.

 


Seleccionar área
