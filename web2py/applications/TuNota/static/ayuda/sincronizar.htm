Sincronizar

 Top  Previous  Next

Para subir las revisiones al sistema web o actualizar los datos en el móvil es necesario sincronizar las dos aplicaciones. Para esto siga los siguientes pasos:

 

1. Ir a la vista Sincronizar:

 

En la pantalla principal:

 

1.1 En la pantalla principal seleccione el cuadro Última sincronización.

 


Cuadro última sincronización

 

En el menú:

 

1.2 Abre el menú principal pulsando el icono de barras y selecciona la opción Sincronizar datos:

 


Abrir menú
         

Sincronizar datos

 

 

2. Después de conectarnos al sistema web, es necesario por seguridad introducir credenciales, estas son las mismas que se utilizan para acceder la web. Luego presiona Continuar.

 


Introducir credenciales

 

3. Presiona el botón Sincronizar:

 


Sincronizar datos

 

4. Si la sincronización se realiza correctamente y no hay cambios el sistema muestra el siguiente mensaje:

 


Sincronización satisfactoria

 

5. En caso de ocurrir cambios en los estados de los medios el sistema lo notifica:

 


Cambios en los medios
