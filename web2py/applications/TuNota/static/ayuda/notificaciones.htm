Existen eventos en el sistema que emiten una notificación a los usuarios que afecten, como actualizar la base de datos o subir una nueva revisión de un área. Para revisar las notificaciones seleccione el icono de campana que se encuentra en la esquina superior derecha del sistema. En caso de existir nuevas notificaciones, la campanita emite una luz roja.

 


Notificaciones

 

Al presionar sobre ella se despliega un contenedor donde se encuentran las nuevas notificaciones, en caso de que no haya se muestra "No existen nuevas notificaciones". Para ver todas las notificaciones seleccione la opción "Ver todas las notificaciones...".
