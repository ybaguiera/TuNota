Generar etiquetas por áreas

 Top  Previous  Next

Se pueden generar etiquetas de un área determinada. Primeramente, selecciona en el menú la opción Etiquetas > Generar por área:

 


Generar etiquetas

 

Luego seleccione el área y presione Enviar:

 


Seleccionar área

 

 

El sistema generará las etiquetas y mostrará las opciones para exportar las etiquetas a PDF. Existen dos opciones, aunque cualquiera de ellas puede ser utilizada, cada una se usa en situaciones diferentes. La opción Generar PDF se utiliza para cuando hay más de 500 etiquetas (esto por cuestiones de rendimiento), y en el caso de Generar PDF usando html2pdf para menos de 500 (esta opción divide los documentos cada 100 etiquetas, pero caben más etiquetas por hoja que en la otra opción).
