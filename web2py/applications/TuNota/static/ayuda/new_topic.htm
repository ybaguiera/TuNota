Cargar datos desde archivo CSV

 Top  Previous  Next

Antes de realizar operaciones sobre los datos de los medios básicos, estos necesitan ser cargados. Primero se necesita el fichero Excel que contiene todos estos datos, el cuál es generado por el Sistema que utiliza el personal de Contabilidad. Como este Sistema no es el mismo para todas las entidades, se requiere que la información sea pre-formateada y luego guardada en un fichero CSV (o Valores separados por coma). No importa si no se puede obtener este fichero, basta con tener la información.

 

Primero se crea un fichero Excel y luego se le adicionan los datos en el siguiente formato:

 

Para la información de cada área se inicia con el nombre del responsable, primero se pone Responsable: y un espacio y luego el nombre.

 

•Primera fila, primera columna: Responsable: NombreDelResponsable.

 


Nombre del responsable

 

Luego, se inserta el nombre del área, primero se escribe Area: (sin tilde), un espacio y luego el nombre:

 

•Segunda fila, primera columna: Area: NombreDelArea.

 


Nombre del área

 

Después se deja un espacio en blanco y se insertan todos los medios de la siguiente manera. Por cada medio se pone, en la primera columna el código y en la segunda columna la descripción.

 

•Primera columna: I-856523
•Segunda columna: Silla plástica

 


Datos de los medios

 

Importante: Ninguno de los datos debe contener el caracter punto y coma ( ; ).

 

Nota: Los medios se pueden adicionar dejando espacios en blanco, pero por organización es preferible que se inserten uno debajo de otro.

 

Para adicionar otra área, se deja un espacio en blanco y luego se repite el proceso anterior con los nuevos datos (no importa que se repita el Responsable, lo importante es que ni el área ni los medios se repitan):

 


Adicionar múltiples datos

 

Después de tener todos los datos guardados en el Excel, guardamos el fichero en formato CSV. En este ejemplo se utiliza el Excel 2016, pero puede utilizarse cualquier procesador de hojas de cálculo siempre y cuando pueda exportar los ficheros en este formato. En caso de utilizar el Excel, se realiza de la siguiente forma:

 

1.Se selecciona Archivo > Guardar como y se selecciona la opción Examinar:

 

 

 

2. Luego se selecciona el directorio donde guardaremos el fichero y el formato CSV, después presionamos la opción Guardar:

 


Guardar archivo CSV

 

 

Importante: Después de guardar el fichero en el formato CSV, no lo abra con el Excel, utilice un editor de texto plano. Ya que al cargarlo en el Excel se pueden perder datos importantes como el 0 delante de un código (09729 -- 9729). Para volver a modificar los datos, utilice el archivo Excel previo y sobrescriba el fichero csv.

 

Después de tener el fichero CSV, vaya al sistema web y seleccione la opción del menú Base de datos > Importar datos:

 


Seleccionar opción Importar Datos

 

Posteriormente presione el botón Examinar y seleccione el fichero csv:

 


Seleccionar archivo CSV

 

Luego seleccione el botón Importar:

 


Importar fichero

 

Si los datos son cargados correctamente el sistema mostrará los cambios realizados en la base de datos con respecto a los datos anteriores, si no mostrará un mensaje de error:
