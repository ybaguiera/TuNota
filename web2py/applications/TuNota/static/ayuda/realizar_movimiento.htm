Para realizar un movimiento a algún medio, selecciona en el menú la opción Medios Básicos > Realizar movimiento.

 


Realizar movimiento

 

Luego de encontrar el medio, selecciona el botón Seleccionar medio que le corresponde en la columna del mismo nombre. Después selecciona el área destino y presiona el botón Continuar.

 


Seleccionar área destino

 

Se mostrará el documento, y una copia del mismo, que se requiere para realizar este procedimiento, y se muestra el botón Generar documento que genera un documento pdf para luego imprimirlo.
